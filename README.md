# MCU Programming Assignments

This repository contains my MCU programming assignments, focusing on low-level programming techniques used in embedded firmware development, including GPIO control, register-level programming, peripheral configuration, and real-time task handling.

The goal is to strengthen my understanding of microcontroller architecture, hardware interaction, and embedded software design through hands-on practice with typical MCU examples.
